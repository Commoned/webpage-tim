Die Idee der Seite war als Produkt ein fast vollständig anpassbaren Tablet-PC anzupreisen.

Die Start-(und Einzige)seite ist index.html.
Der Ordner assets umfasst alle nötigen Bilder und css/javascript dateien die für das korrekte Anzeigen der Seite nötig sind.

der Darkmode Button stellt den Darkmode der Seite ein.

Die links in der Navbar führen auf Textanker innnerhalb der Seite wenn mehr Seiten gefordert wären würden sie auf andere Seiten zeigen.

Eyecatcher:

- Navbar schrumpft beim scrollen
- Banner im Hintergrund wie Fenster
- Darkmode
- Bilder die beim Hovern größer werden

Eastereggs(sind nicht gefährlich, Sie können gern im Code nachschauen):

- 10 Mal auf Windows Icon klicken
- 10 Mal auf Apple Icon klicken
